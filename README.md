# Site vitrine – Web Designer / Créateur de sites

## Objectif
Créer un site vitrine professionnel pour présenter mon activité de création de sites web et de refonte de sites existants.

## Spécifications
- Site one-page responsive, rapide, clair
- Sections : Hero, Services, Process, Portfolio, Offres, FAQ, Contact, Footer
- HTML + CSS (fichier séparé)
- JavaScript uniquement dans le HTML (en bas, avant `</body>`)
- Design moderne (fond clair, cartes, boutons)
- Animations légères : menu mobile, scroll fluide, FAQ accordéon
- Déploiement via GitHub Pages
- Projet versionné avec Git et poussé sur GitHub
